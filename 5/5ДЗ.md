# Проверка того, что код, дуальный коду Хэмминга, является симплекс-кодом

## 1. Постановка задачи

Нужно проверить, что бинарный код, дуальный коду Хэмминга, действительно совпадает с бинарным симплекс-кодом. Для этого напишем программу, которая:

1. строит проверочную матрицу `H` кода Хэмминга;
2. рассматривает линейную оболочку строк `H`, то есть дуальный код `C⊥`;
3. вычисляет его основные параметры;
4. проверяет, что они совпадают с параметрами симплекс-кода.

---

## 2. Теоретическая основа

Бинарный код Хэмминга имеет параметры:

`[2^m - 1; 2^m - 1 - m; 3]`.

Его проверочная матрица `H` состоит из **всех ненулевых двоичных столбцов длины** `m`.

Если `C` — код Хэмминга, то его дуальный код `C⊥` порождается строками матрицы `H`:

`C⊥ = <rows(H)>`.

С другой стороны, бинарный симплекс-код определяется как код с параметрами:

`[2^m - 1; m; 2^(m-1)]`.

Порождающая матрица симплекс-кода также состоит из всех ненулевых двоичных столбцов длины `m`.

Поэтому уже из определения видно, что дуальный код кода Хэмминга и есть симплекс-код. Программа ниже вычисляет величины, которые это подтверждают на практике:

- длину `n`;
- размерность дуального кода;
- число кодовых слов;
- веса кодовых слов;
- минимальное расстояние.

Если для `C⊥` выполняется:

- `n = 2^m - 1`,
- `k = m`,
- `|C⊥| = 2^m`,
- все ненулевые кодовые слова имеют вес `2^(m-1)`,
- минимальное расстояние равно `2^(m-1)`,

то мы получаем ровно параметры симплекс-кода.

---

## 3. Идея программы

Программа работает так:

1. Для заданного `m` строит матрицу `H`, столбцы которой — все ненулевые векторы из `F_2^m`.
2. Вычисляет ранг `H` над `F_2`.
3. Строит все линейные комбинации строк `H`, то есть все слова дуального кода.
4. Считает веса всех кодовых слов.
5. Находит минимальное расстояние и сравнивает полученные параметры с параметрами симплекс-кода.

---

## 4. Основные фрагменты программы

### 4.1. Построение проверочной матрицы кода Хэмминга

```python

def int_to_bits(x: int, m: int):
    return [(x >> i) & 1 for i in range(m - 1, -1, -1)]


def build_hamming_parity_check_matrix(m: int):
    """Стандартная проверочная матрица H для бинарного кода Хэмминга.
    Ее столбцы — все ненулевые двоичные векторы длины m.
    """
    n = 2 ** m - 1
    cols = [int_to_bits(i, m) for i in range(1, n + 1)]
    H = [[cols[j][i] for j in range(n)] for i in range(m)]
    return H
```

### Пояснение

Функция `build_hamming_parity_check_matrix(m)` строит матрицу `H` размера `m × (2^m - 1)`. Именно такая матрица используется для кода Хэмминга. Так как дуальный код порождается строками `H`, эта же матрица фактически становится порождающей матрицей для `C⊥`.

---

### 4.2. Вычисление ранга над `F_2`

```python

def rank_mod2(matrix):
    A = [row[:] for row in matrix]
    rows = len(A)
    cols = len(A[0])
    r = 0

    for c in range(cols):
        pivot = None
        for i in range(r, rows):
            if A[i][c] == 1:
                pivot = i
                break
        if pivot is None:
            continue

        A[r], A[pivot] = A[pivot], A[r]

        for i in range(rows):
            if i != r and A[i][c] == 1:
                A[i] = [a ^ b for a, b in zip(A[i], A[r])]

        r += 1
        if r == rows:
            break

    return r
```

### Пояснение

Здесь реализован обычный метод Гаусса, но все операции выполняются по модулю 2. Ранг матрицы `H` равен размерности дуального кода. Для симплекс-кода должна получиться размерность `m`.

---

### 4.3. Построение всех кодовых слов дуального кода

```python
from itertools import product


def all_linear_combinations(rows):
    k = len(rows)
    n = len(rows[0]) if rows else 0
    codewords = []

    for coeffs in product([0, 1], repeat=k):
        word = [0] * n
        for c, row in zip(coeffs, rows):
            if c:
                word = [a ^ b for a, b in zip(word, row)]
        codewords.append(tuple(word))

    return codewords
```

### Пояснение

Если строки матрицы `H` являются генераторами дуального кода, то все кодовые слова получаются как их линейные комбинации. Всего таких комбинаций должно быть `2^m`, если размерность кода равна `m`.

---

### 4.4. Анализ параметров дуального кода

```python

def weight(v):
    return sum(v)


def hamming_distance(u, v):
    return sum(a ^ b for a, b in zip(u, v))


def analyze_dual_of_hamming(m: int):
    H = build_hamming_parity_check_matrix(m)
    n = len(H[0])
    k_dual = rank_mod2(H)
    dual_code = all_linear_combinations(H)
    unique_words = sorted(set(dual_code))

    weights = sorted(weight(v) for v in unique_words)
    nonzero_weights = sorted({w for w in weights if w > 0})

    distances = sorted({
        hamming_distance(unique_words[i], unique_words[j])
        for i in range(len(unique_words))
        for j in range(i + 1, len(unique_words))
    })
    nonzero_distances = [d for d in distances if d > 0]

    return {
        'm': m,
        'n': n,
        'rank_H': k_dual,
        'dual_size': len(unique_words),
        'nonzero_weights': nonzero_weights,
        'min_distance': min(nonzero_distances),
    }
```

### Пояснение

Эта функция вычисляет все основные характеристики дуального кода:

- длину `n`;
- размерность `rank(H)`;
- число кодовых слов `|C⊥|`;
- множество ненулевых весов;
- минимальное расстояние.

Для симплекс-кода ожидается, что все ненулевые веса совпадают и равны `2^(m-1)`.

## 5. Полный текст программы

```python
from itertools import product


def int_to_bits(x: int, m: int):
    return [(x >> i) & 1 for i in range(m - 1, -1, -1)]


def build_hamming_parity_check_matrix(m: int):
    """Return the standard parity-check matrix H for the binary Hamming code.
    Columns are all nonzero binary vectors of length m.
    """
    n = 2 ** m - 1
    cols = [int_to_bits(i, m) for i in range(1, n + 1)]
    H = [[cols[j][i] for j in range(n)] for i in range(m)]
    return H


def rank_mod2(matrix):
    A = [row[:] for row in matrix]
    if not A:
        return 0
    rows = len(A)
    cols = len(A[0])
    r = 0

    for c in range(cols):
        pivot = None
        for i in range(r, rows):
            if A[i][c] == 1:
                pivot = i
                break
        if pivot is None:
            continue
        A[r], A[pivot] = A[pivot], A[r]
        for i in range(rows):
            if i != r and A[i][c] == 1:
                A[i] = [(a ^ b) for a, b in zip(A[i], A[r])]
        r += 1
        if r == rows:
            break
    return r


def all_linear_combinations(rows):
    k = len(rows)
    n = len(rows[0]) if rows else 0
    codewords = []
    for coeffs in product([0, 1], repeat=k):
        word = [0] * n
        for c, row in zip(coeffs, rows):
            if c:
                word = [a ^ b for a, b in zip(word, row)]
        codewords.append(tuple(word))
    return codewords


def weight(v):
    return sum(v)


def hamming_distance(u, v):
    return sum(a ^ b for a, b in zip(u, v))


def analyze_dual_of_hamming(m: int):
    H = build_hamming_parity_check_matrix(m)
    n = len(H[0])
    k_dual = rank_mod2(H)
    dual_code = all_linear_combinations(H)
    unique_words = sorted(set(dual_code))
    weights = sorted(weight(v) for v in unique_words)
    nonzero_weights = sorted({w for w in weights if w > 0})
    distances = sorted({hamming_distance(unique_words[i], unique_words[j])
                        for i in range(len(unique_words))
                        for j in range(i + 1, len(unique_words))})
    nonzero_distances = [d for d in distances if d > 0]

    return {
        'm': m,
        'n': n,
        'H': H,
        'rank_H': k_dual,
        'dual_size': len(unique_words),
        'weights': weights,
        'nonzero_weights': nonzero_weights,
        'min_distance': min(nonzero_distances) if nonzero_distances else 0,
        'all_nonzero_distances': nonzero_distances,
    }


def format_matrix(H):
    return '\n'.join(' '.join(map(str, row)) for row in H)


def main():
    for m in [3, 4, 5]:
        res = analyze_dual_of_hamming(m)
        print('=' * 72)
        print(f"m = {res['m']}")
        print(f"n = 2^{res['m']} - 1 = {res['n']}")
        print('Матрица H:')
        print(format_matrix(res['H']))
        print(f"rank(H) = {res['rank_H']}")
        print(f"|C^perp| = {res['dual_size']} = 2^{res['rank_H']}")
        print(f"Ненулевые веса кодовых слов: {res['nonzero_weights']}")
        print(f"Минимальное расстояние: {res['min_distance']}")
        print(f"Все ненулевые попарные расстояния: {sorted(set(res['all_nonzero_distances']))}")
        print(f"Ожидаемые параметры симплекс-кода: [{res['n']}, {res['m']}, {2 ** (res['m'] - 1)}]")
        ok = (
            res['rank_H'] == m and
            res['dual_size'] == 2 ** m and
            res['nonzero_weights'] == [2 ** (m - 1)] and
            res['min_distance'] == 2 ** (m - 1)
        )
        print('Проверка:', 'OK, дуальный код совпадает с бинарным симплекс-кодом.' if ok else 'Проверка НЕ пройдена.')


if __name__ == '__main__':
    main()
```

## 8. Итоговый вывод

Мы использовали тот факт, что:

1. проверочная матрица кода Хэмминга содержит все ненулевые двоичные столбцы длины `m`;
2. дуальный код порождается строками этой матрицы;
3. симплекс-код определяется той же самой матрицей как порождающей.

Следовательно, **код, дуальный коду Хэмминга, действительно является бинарным симплекс-кодом**.

Написанная программа вычисляет необходимые величины и подтверждает это по параметрам:

`[2^m - 1; m; 2^(m-1)]`.

То есть для бинарного кода Хэмминга справедливо:

`(H_m)^⊥ = S_m`,

где `H_m` — код Хэмминга, а `S_m` — симплекс-код.
